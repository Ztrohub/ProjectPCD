Let's Plot It !

First, insert your Rubik's Cube image by pressing the Choose file
Then select any options you want to use by checking the boxes.

Don't forget to press the Process button.
You may use Compare feature below to show some results.

Isometric Color

Show color segmentation from your Rubik's Cube image and display it like your original image

Image Lines

Show lines of image. The lines are usefull for side-by-side segmentation.

Pattern Correlation Value

Show surface value of the image that correlate against some features.

Projected Image

Show each side of the Rubik's Cube that already projected into a cube's nets.

Projected Color

Show recognition color on each side side of the Rubik's Cube and from the cube's nets.

3D Plot

Show scatter plot of recognition color in 3D Plot.
